A simplistic tutorial


Okay, disclaimer. This website isn't half as good as some of the ones I looked at and learned from. Now since I am a beginner, the structuring of this website is rather poor, and I had no backend support and so there have been a couple cheats here and there. Plus, if you have zero web development experience, you might as well leave this page.

Now that that's out of the way, let's dive straight into it. Step 1, learn HTML and CSS from w3schools. I won't link them here because I'm jealous of how popular they are. Step 2, learn javascript syntax. This shouldn't be too difficult if you are familiar with programming.

An ordinary tutorial would now ask you to practice javascript and learn bootstrap. No, thanks! We are short on time and so simply google the stuff you want. Need a navbar, google "navbar bootsrap", hop into the first link and copy the code of whichever navbar style you like. If you really understood HTML and CSS, the code will make sense automatically and you should be able to figure hacks out, with occasional help from stackoverflow. I mean, I was able to create everything you see right now using exactly this method. Stuff I didn't like, I changed using javascript/Css and boy was it easy.

It does hurt when all of a sudden you are not allowed to use jQuery and it might take an entire day to find jQuery alternatives in vanilla javascript and you might have to drop/replace important features but it's part of the process, I guess. That being said, if you have your gray cells turned on and aren't hesitant to copy paste code, website development is definitely a piece of cake.
